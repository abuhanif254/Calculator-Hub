## Qu'est-ce qu'un QR Code ?

Un **QR Code (Quick Response Code)** est un code-barres matriciel en 2D (deux dimensions) qui stocke des données binaires dans une grille de carrés noirs et blancs.

Contrairement aux codes-barres traditionnels des supermarchés, qui ne stockent que quelques chiffres à l'horizontale, un QR code stocke les informations à la fois verticalement et horizontalement. Il peut ainsi contenir jusqu'à **4 296 caractères alphanumériques**. Un simple scan via l'appareil photo du smartphone traduit instantanément la grille en une page Web, un contact ou un réseau WiFi.

---

## Anatomie d'un Code QR (Comment ça marche ?)

Un QR code n'est pas un motif aléatoire. Il suit une norme internationale (ISO/IEC 18004) et contient plusieurs balises essentielles :

1. **Les motifs de recherche (Les 3 gros carrés) :** Ils permettent à l'appareil photo de repérer immédiatement la position du code et son sens de rotation. C'est pourquoi un QR Code peut être scanné de travers ou à l'envers.
2. **Les motifs d'alignement :** De petits carrés internes qui aident l'objectif à corriger les distorsions si le code est imprimé sur une surface incurvée (comme une canette).
3. **La Zone Silencieuse (Quiet Zone) :** Une marge vide obligatoire autour du code (idéalement de la largeur de 4 petits carrés). Sans elle, le scanner confond le code avec le texte environnant.

---

## Le Niveau de Correction d'Erreurs (Reed-Solomon)

C'est la magie du QR code : sa capacité d'auto-réparation. Grâce à un algorithme mathématique (Reed-Solomon), le scanner peut reconstruire le message même si une partie du papier est déchirée ou sale.

Il y a 4 niveaux :
* **L (Faible - 7%)** : Idéal pour les écrans propres, le motif reste très simple et épuré.
* **M (Moyen - 15%)** : Le standard habituel.
* **Q (Quartile - 25%)** : Pour les affiches en extérieur.
* **H (Haut - 30%)** : **Ce niveau est obligatoire pour placer un logo**. Le scanner considérera le logo comme un \"dégât\" sur le QR code et utilisera la redondance des 30% d'erreurs pour deviner les données cachées.

## Bonnes pratiques pour l'impression (Vectoriel)

Pour que vos QR codes personnalisés aient un rendu professionnel :

* **Un contraste élevé** : Le motif doit être plus foncé que le fond. Évitez le jaune clair sur fond blanc.
* **Formats de téléchargement** : Utilisez le **SVG** pour tous vos travaux d'impression (Bâches, Affiches, Dépliants, Illustrator). Le SVG ne se pixellise jamais, peu importe sa taille. L'image restera extrêmement nette et l'imprimeur vous remerciera. Conservez le **PNG** pour les sites web ou les emails.

### Cas d'usages courants (Payloads)

1. **Réseau WiFi** : Code de la forme `WIFI:T:WPA;S:MonReseau;P:MonMotDePasse;;`. Les invités scannent le code avec un iPhone ou Android et se connectent en un clic, sans épeler un mot de passe complexe.
2. **Contact vCard** : Encode une fiche de contact complète. Un bouton \"Ajouter au répertoire\" apparaîtra sur l'écran.
3. **Envoi de SMS** : Un code peut ouvrir l'application Message du smartphone et pré-remplir le numéro et le texte (`SMSTO:+33612345678:Je souhaite m'inscrire`). Idéal pour l'acquisition de clients.
