## Was ist ein QR-Code?

Ein **QR-Code (Quick Response Code)** ist ein zweidimensionaler Barcode (Matrix). Er wurde ursprünglich in der japanischen Automobilindustrie erfunden, um Bauteile schneller scannen zu können.

Im Gegensatz zu alten Strichcodes aus dem Supermarkt, die nur wenige Zahlen horizontal speichern können, speichern QR-Codes die Daten sowohl vertikal als auch horizontal. Dadurch kann ein einziger QR-Code bis zu **4.296 alphanumerische Zeichen** enthalten. 

Für den Endnutzer funktioniert ein QR-Code wie ein extrem schneller, gedruckter Hyperlink. Die Smartphone-Kamera übersetzt die schwarzen und weißen Quadrate blitzschnell zurück in Text, Webadressen oder WLAN-Daten.

---

## Wie funktioniert ein QR-Code?

Das scheinbare Chaos aus Pixeln folgt in Wahrheit strengen Mustern (ISO/IEC 18004 Norm):

1. **Positionsmarkierungen (Die 3 großen Quadrate):** Sie befinden sich in den Ecken. Dadurch weiß der Kamera-Scanner sofort, wie groß der Code ist und wie er gedreht wurde. Sie können einen QR-Code also problemlos kopfüber scannen.
2. **Ausrichtungsmuster (Alignment Patterns):** Kleine Hilfs-Quadrate im Inneren, die dem Smartphone helfen, den Code zu lesen, wenn er auf einer runden Fläche (z.B. einer Getränkedose) gedruckt wurde.
3. **Die Ruhezone (Quiet Zone):** Jeder QR-Code benötigt einen leeren Rand um den Code herum. Ohne diesen Rand würde die Scanner-App den Code mit dem umliegenden Text oder den Farben des Flyers verwechseln.

---

## Fehlerkorrektur (Reed-Solomon) & Logos

Das genialste Feature an QR-Codes ist ihre Fehlertoleranz. Dank eines mathematischen Algorithmus (Reed-Solomon-Korrektur) kann der Code selbst dann gelesen werden, wenn er beschmiert, zerkratzt oder teilweise verdeckt ist.

Es gibt vier Stufen der Fehlerkorrektur:
* **L (Low - 7%)**: Ergibt den saubersten, am einfachsten lesbaren Code.
* **M (Medium - 15%)**: Der Standard für normale Ausdrucke.
* **Q (Quartile - 25%)**: Für Codes, die oft angefasst werden.
* **H (High - 30%)**: Das höchste Level.

**Warum das für Designer wichtig ist:** Wenn Sie ein **Firmenlogo** in die Mitte des QR-Codes setzen, verdeckt dieses Bild die Datenblöcke. Die Scanner-App behandelt das Logo wie einen \"Fleck\". Sie müssen daher zwingend das Level **H (30%)** auswählen. Der Algorithmus nutzt dann die doppelt vorhandenen, verstreuten Rest-Daten im Code, um den fehlenden Text wiederherzustellen.

## Tipps für den perfekten Vektor-Druck

Wenn Sie den QR-Code professionell für Marketingmaterialien drucken wollen:

1. **Vermeiden Sie PNGs:** Ein Pixelbild (PNG oder JPG) wird unscharf, wenn die Druckerei es vergrößert. Laden Sie stattdessen immer eine **SVG-Datei (Vektor)** aus unserem Generator herunter. SVG-Bilder berechnen Kanten mathematisch und sind bei jeder Größe gestochen scharf.
2. **Kontrast wahren:** Das Muster muss immer deutlich dunkler sein als der Hintergrund.
3. **Ruhezone beachten:** Schneiden Sie den Rand des Codes nicht ab.

### Beliebte Code-Typen (Payloads)

1. **WLAN (Wi-Fi):** Der Aufbau `WIFI:T:WPA;S:MeinNetz;P:Geheim;` ermöglicht es iOS und Android, das WLAN ohne Eintippen des Passworts einzurichten.
2. **Kalenderereignis (iCalendar):** Speichert eine Einladung, die der Besucher sofort in seinen Google- oder Apple-Kalender übernehmen kann.
3. **SMS und WhatsApp:** Startet die Nachrichten-App und trägt die Zielnummer sowie einen vorgefertigten Text direkt ein. Ideal für Tischreservierungen!
